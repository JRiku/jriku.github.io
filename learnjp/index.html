Short timeline to basic fluency

Italic means optional stuff you can do if you want to

Learn Hiragana & Katakana (2 Weeks (reading only) to 6 weeks (learning handwriting))

→ Learn Hiragana and Katakana with Tofugu
→ Hiragana and Katakana game
→ Handwriting worksheets (Hiragana, Katakana)

Learn basic Grammar (~2 weeks while starting to learn vocabulary at the same time)

→ Tae Kim's guide to Japanese grammar (p.29-81)

Learn Vocabulary

→ Install Anki
→ Install Yomitan to mine new words you encounter
→ Do the Kaishi 1.5k deck for Anki (300 days if you do 5 new cards a day)

Learn advanced grammar

→ Do Bunpro for Grammar (if you have Money) → → Continue with Tae Kim's guide to Japanese grammar
